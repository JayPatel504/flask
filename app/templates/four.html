  • What is Flask?
    • A micro web framework
      • Micro meaning Flask is simple and does not rely on particular tools or libraries
      • It has no database abstraction layer, form validation, et cetera
      • In other words, Flask does not make any decisions
    • However, Flask supports extensions to add such functionality like database integration, form validation, upload handling, various open authentication technologies, and more
    • Click here to learn more
  • Flask depends on the Jinja template engine and the Werkzeug WSGI toolkit
    • What is Jinja?
      • Full-featured template engine
      • Click here to learn more
    • What is WSGI?
      • It is a Web Server Gateway Interface that describes the communication between web server and web app, and how web apps can be chained together
